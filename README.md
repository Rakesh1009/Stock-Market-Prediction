CS337 Course Project: Stock Market Predction

-----------------------------------------------------------------

The basic directory files are:
>Input-contains the datasets, for instance we used IBM.
>Output-png files of the output graphs
>Src-all the python files for the project

We tried 4 types of models namely,

1. Linear-ReLU-Linear
2. Long short-term memory (LSTM)
3. Linear-ReLU-LSTM
4. Gated recurrent units (GRU)

The code for these models are in 
"lrl.py"
"lstm.py"
"lrlstm.py"
"gru.py"
files respectively.

-----------------------------------------------------------------
Command to run the model

python3 <model_name>.py
